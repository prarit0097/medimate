MediMate API

Backend is running. This project powers patient, caregiver, medication, reminder, dose tracking, and provider-lite workflows for the MediMate web app and future Flutter client.

Current build is API-first, so the main interactive surfaces right now are the schema docs, admin, and authenticated API endpoints.
